# This is my prototype website.
This website was created during the Frontend Simplified Bootcamp. This website displays fundamental skills in HTML and CSS. It focuses on best practices for creating websites. During this project I created a responsive website with a navigation bar, header, main content sections, and a footer. I used semantic HTML elements to structure the content and applied CSS for styling and layout. The website is designed to be visually appealing and user-friendly, with a focus on clear typography, consistent spacing, and intuitive navigation.
